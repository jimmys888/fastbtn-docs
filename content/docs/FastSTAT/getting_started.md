## Installation
Follow these steps to install **FastSTAT** on MetaTrader 5:

1. Extract the `faststat.zip` file (or open it directly).
2. Locate the **`FastSTAT.ex5`** file.
3. Copy `FastSTAT.ex5`.

4. Open MetaTrader 5:
   - Click **File â†’ Open Data Folder**
   - Or press `Ctrl + Shift + D`

5. Open the **`MQL5`** folder, then open **`Indicators`**.

6. Paste `FastSTAT.ex5` into the **Indicators** folder.

7. Return to MetaTrader 5:
   - Open the **Navigator** panel (`Ctrl + N`)
   - Right-click **Indicators**
   - Click **Refresh**

You should now see **FastSTAT** listed under *Indicators*.

---

## Attaching FastSTAT to a Chart

1. Open the chart where you want FastSTAT to appear.
2. Drag **FastSTAT** onto the chart or double-click it.
3. Adjust the input settings if needed.
4. Click **OK**.

FastSTAT information will appear on the chart immediately.
